## Instalacja

Aby uruchomić ten projekt, wykonaj następujący skrypt

1. Sklonuj repozytorium:

    ```bash
    git clone git@github.com:DamianJudek/SNiFi.git
    ```

2. Przejdź do katalogu projektu:

    ```bash
    cd SNiFi
    ```

3. Zainstaluj zależności:

    ```bash
    python ./setup.py
    ```
